What is BlaziNG ? BlaziNG is a User Interface (UI) component library whose functionality is to facilitate the construction of attractive, consistent and functional web pages or web applications Features
  • 30+ high-quality Angular components out of the box.
  • Written in TypeScript with complete defined types.
  • Support OnPush mode, high performance.
  • Powerful theme customization in every detail.
